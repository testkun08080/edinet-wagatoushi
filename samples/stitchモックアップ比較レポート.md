作成日: 2025-02-16

---

## 1. サンプル構成の概要

| フォルダ | 内容 | テーマ |
|----------|------|--------|
| `samples/全体検索/` | **アプリ全体のレイアウト・デザインの参考**（サイドバー＋メイン、ライトテーマ） | ライトモード |
| `samples/各ページ/` | 企業詳細ページ（日本語・ダーク/ターミナル風） | ダークモード |

※ **企業一覧画面**は `samples/全体検索/` のレイアウト（サイドバー、ヘッダー、配色、余白など）を参考にする。

### 2.3 モックアップにあって現行データに「ない」項目（削除 or 別ソース）

モックアップのテーブルで使用しているが、`company_metrics.json` や `summaries/*.json` に**直接存在しない**項目:

| モックアップ表記 | 現行データ | 対応 |
|------------------|------------|------|
| 売上総利益 | summary に direct にはなし / pl の「売上総利益又は売上総損失」 | pl から取得可能 |
| 販管費（販売費及び一般管理費） | pl | pl から取得可能 |
| 営業外損益 | pl の営業外収益−営業外費用 | 計算で導出可能 |
| 資産合計 | summary「総資産額」 / bs「総資産」 | あり |
| 現金同等物 | summary「現金及び現金同等物の残高」 | あり |
| LTM 平均 | — | 計算で導出（現状未実装） |

---

### 2.4 現行にあるがモックアップに表示していない指標

現行 `ColumnVisibilityContext` / `company_metrics.json` にあるが、モックアップのヘッダーKPI・テーブルには**表示していない**指標:

- 計算日 / 決算月
- EDINETコード
- 配当性向
- EPS / BPS
- 1株当たり配当金 / 発行済株式総数
- 営業利益率 / 純利益率（算出指標）
- 流動資産 / 流動負債 / 負債
- 純資産額 / 総資産額
- 投資有価証券
- 営業CF / 投資CF / 財務CF

※ 現行デザインを正とする場合、これらは**表示すべきパラメーター**として維持する。

## 5. いらない・調整すべき要素

1. **時価総額 / Market Cap**: **非表示**とする（株価連携までデータなし）。
2. **LTM 平均列**: 現行に未実装。実装するかどうか要検討。
3. **競合比較タブ**: 現行に未実装。新規機能として要検討。
4. **四半期報告書タブ**: 現行は「経営成績」「損益」「貸借」「CF」の4セクション。タブ名・構成のすり合わせが必要。
5. **銘柄コード表記**: 各ページは `1376:JP`、現行は `13760`（4桁／5桁の違い）。現行の secCode が正。

## 6. 足りない要素（現行にあり、モックアップにない）

1. **企業一覧画面**: `samples/全体検索/` のレイアウト・デザインを参考に実装する（サイドバー＋メインエリアの構成）。
2. **経営成績サマリ**: モックアップは「売上高・経常利益・当期純利益・自己資本比率・営業CF」等の一部のみ。現行の `summary` 全項目との対応が未整理。
3. **列表示切替（ColumnVisibilityControls）**: 34項目の表示/非表示。モックアップには未反映。
4. **フィルター**: PER・PBR・配当利回り等による絞り込み。モックアップには未反映。
5. **お気に入り（Watchlist）**: ボタンはあるが、一覧との連携UIはモックアップにない。

## 9. 検索項目の齟齬確認（Stitch vs 現行）

現行の検索・フィルター項目をキープする前提で、Stitch 構成との整合性を確認した。

### 現行の検索・フィルター項目（キープ対象）

| カテゴリ | 項目 | データソース |
|----------|------|--------------|
| **検索** | 会社名 | `filerName` |
| | 銘柄コード | `secCode` |
| | お気に入りのみ | FavoritesContext |
| **基本フィルター** | 自己資本比率（最小・最大） | `自己資本比率` |
| | EPS（最小・最大） | `EPS` |
| | 売上高（最小・最大、百万円） | `売上高` |
| | ROE（最小・最大） | `ROE` |
| | 総資産額（最小・最大、百万円） | `総資産額` |

### Stitch 全体検索サイドバーの内容

- ナビ: Dashboard / Company Analysis / Watchlist
- Recent Searches: 直近閲覧企業へのリンク（例: カネコ種苗, 任天堂）

**→ 検索・フィルター用の入力欄・チェックボックスは Stitch モックアップに存在しない。**

### 齟齬と対応方針

| 齟齬 | 対応 |
|------|------|
| Stitch には検索（会社名・銘柄コード）がない | 現行の検索 UI をサイドバーに**追加してキープ** |
| Stitch には基本フィルター（自己資本比率、EPS、売上高、ROE、総資産額）がない | 現行のフィルター UI をサイドバーに**追加してキープ** |
| Stitch には「お気に入りのみ」チェックがない | 現行のチェックボックスを**追加してキープ** |
| Stitch の「Recent Searches」は現行にない | Stitch に合わせて **Recent Searches を追加**可能（直近閲覧企業のリンク） |

**結論**: 齟齬あり。Stitch のレイアウトを採用する場合、サイドバーに**現行の検索・フィルター項目をそのまま組み込む**必要がある。Stitch モックアップは検索・フィルターを含んでいないため、現行の `CompanySidebar` の検索・フィルター部分を Stitch 風の見た目に整えつつ維持する形になる。
