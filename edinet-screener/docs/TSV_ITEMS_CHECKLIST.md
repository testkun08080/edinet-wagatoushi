# TSV 項目一覧（判断用）

実際の TSV に含まれる項目をまとめ、利用可否を判断しやすくしたチェックリストです。

---

## 凡例


| 記号  | 意味                                             |
| --- | ---------------------------------------------- |
| ○   | すでにパーサーで抽出済み（summaries / company_metrics で利用可） |
| △   | element_id_table に追加すれば抽出可能                    |
| ×   | TSV に無い or 株価等の外部データが必要                        |

## 8. 株価連携が必要な項目（TSV に無い）・割安指標


| 項目             | 算出式                     | 備考                                                         |
| -------------- | ----------------------- | ---------------------------------------------------------- |
| PER            | 株価 / EPS                | 株価取得が必要                                                    |
| PBR            | 株価 / BPS                | 株価取得が必要                                                    |
| 配当利回り          | 配当額×4 / 株価              | 株価取得が必要                                                    |
| 時価総額           | 株価 × 発行済株式総数            | 株価取得が必要                                                    |
| **ネットキャッシュ**   | （流動資産 ＋ 投資有価証券×70％）－ 負債 | **EDINET のみで算出可**（流動資産・投資有価証券・負債は TSV/company_metrics にあり） |
| **ネットキャッシュ比率** | ネットキャッシュ / 時価総額         | 時価総額が必要。**J-Quants はオミット**のため未算出。割安判断用。                    |

## 判断の目安（優先度案）


| 優先度   | 項目            | 用途               |
| ----- | ------------- | ---------------- |
| **高** | 上場市場          | マザーズ・プライム等のフィルター |
| **高** | 業種コード         | 業種フィルター          |
| **高** | 営業利益          | 営業利益率・収益性        |
| **高** | ROE           | 収益性              |
| **高** | 流動資産・流動負債     | 流動比率             |
| **高** | 営業CF          | キャッシュ            |
| **中** | 総資産額          | 規模               |
| **中** | BPS・発行済株式総数   | 株価連携時の準備         |
| **中** | 配当性向・１株当たり配当額 | 配当分析             |
| **中** | セグメント別売上      | セグメント分析          |
| **低** | 大株主情報         | 詳細分析             |
| **低** | 人件費内訳         | 収益性の補助           |
